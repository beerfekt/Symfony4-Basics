## SET UP THE PROJECT (Tested with Ubuntu(Linux))
  
1. Clone the Repo: 
  
    git clone 
 
2. Go into the cloned-folder:
   
     cd homestead_test  
     
 
3. Give execute permission to the script:  

    chmod +x setup.sh

4. Run the script:  
  
    ./setup.sh
